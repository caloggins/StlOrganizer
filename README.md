# Stl Organizer

Helps to organize the zip archives STLs come in.

## Usage

1.  **Select Directory**: Click the "Select Directory" button to choose the folder containing your STL zip archives. The selected path will be displayed next to the button.
2.  **Select Operation**: Choose the desired action from the drop-down menu (e.g., specific organization or processing tasks).
3.  **Execute Operation**: Click the button at the bottom right (labeled with your selected operation) to start the process.
4.  **Monitor Progress**: A progress bar will appear while the operation is running, and status messages will be displayed in the center of the window.
5.  **Cancel**: If you need to stop an operation, click the "Cancel" button.
